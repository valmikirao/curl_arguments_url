============
Installation
============


Stable release
--------------

To install Curl with Arguments for Url, run this command in your terminal:

.. code-block:: console

    $ pip install curl_arguments_url

This is the preferred method to install Curl with Arguments for Url, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Curl with Arguments for Url can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/valmikirao/curl_arguments_url

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/valmikirao/curl_arguments_url/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/valmikirao/curl_arguments_url
.. _tarball: https://github.com/valmikirao/curl_arguments_url/tarball/master
